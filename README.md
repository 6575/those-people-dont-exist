# Launch / Запуск
1. Install poetry / Установите poetry
https://python-poetry.org/docs/#installation

2. Go to the project's root directory / Перейдите в корневую дерикторию проекта

3. Activate the virtual environment / Активируйте виртуальную среду
```sh
poetry env activate
```

4. Install the dependencies / Установите зависимости
```sh
poetry install
```

5. Launch the program / Запустите программу
```sh
poetry run start
```
